# study-gopl
書籍「プログラミング言語Go」について学習するリポジトリ。 
https://github.com/adonovan/gopl.io

## 学習の進め方
- 書籍を読みすすめる。
- 学習した内容は、適宜各章のREADME.mdにメモする。
- サンプルソースは必要に応じて適宜写経する。
- 練習問題はすべて解く。
  - 一旦自力で解いたあと、本書訳者でもある[柴田芳樹氏の回答](https://github.com/YoshikiShibata/gpl)を参考に、解き直す
